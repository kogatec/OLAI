# OLAI (オーライ) - AIバーチャル事務員サービス

中小建設業向けAIバーチャル事務員サービス。LINEをメインインターフェースとして、職人ネットワーク型の小規模事業者（1〜10名）の事務作業を自動化します。

## サービス概要

OLAIは「AIバーチャル事務員」をLINEで提供する、中小建設業向けSaaSです。クロス屋・塗装・電気工事・防水・ハウスクリーニングなど、職人ネットワーク型の小規模事業者が毎月直面する「誰にいくら払うか」「見積・請求書の作成」「現場の進捗把握」をLINEひとつで解決します。

## 主な機能

- LINE連携・メッセージング（@メンション専用設計）
- ユーザー管理（Web登録画面4ステップ、既存PDF取り込み）
- 職人・現場管理（職人登録・元請け登録・現場作成・完了報告解析・単価設定・月次支払い集計）
- 書類作成（見積書・請求書・領収書PDF自動生成・コンビニ印刷ガイド）
- 顧客・名刺管理（名刺OCR自動登録・Eight・Sansan API連携）
- 決済（Stripe・PayPay for Business）
- 紹介・キャンペーン制度
- 管理者ダッシュボード

## 技術スタック

- AIエンジン: Claude Haiku API
- メッセージング: LINE Messaging API, WhatsApp Business API
- OCR: Google Document AI
- DB: PostgreSQL（Supabase）
- ストレージ: Supabase Storage
- PDF生成: Puppeteer / PDFKit
- フロント: Next.js（Vercel）
- バックエンド: Node.js + Railway
- 決済: Stripe, PayPay for Business
