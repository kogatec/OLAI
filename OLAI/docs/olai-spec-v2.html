サービス概要

OLAIは「AIバーチャル事務員」をLINEで提供する、中小建設業向けSaaSだ。クロス屋・塗装・電気工事・防水・ハウスクリーニングなど、職人ネットワーク型の小規模事業者（1〜10名）が毎月直面する「誰にいくら払うか」「見積・請求書の作成」「現場の進捗把握」をLINEひとつで解決する。

項目内容
サービス名OLAI（オーライ）
意味OL（事務員）→ AI。「了解」の音も兼ねる
ドメインolai.help
メインコピー事務員、雇う前に。
サブコピーAIと、いつものLINEで指示するだけ。
月額¥14,800〜（社長プラン）
初期費用¥0
無料期間14日間（紹介制度で最大3ヶ月延長）

解決する3つの課題

課題現状OLAIの解決策
職人への支払い管理電卓・Excelで手動集計LINEの完了報告を自動集計
見積・請求書作成手書き・テンプレートを手動編集LINEで指示→PDF自動生成
現場の進捗把握電話・LINEで個別確認グループ内@メンションで自動記録

ブランド・ロゴ

ロゴアニメーション仕様（初回起動のみ）

アプリ・サービスの初回起動時のみアニメーションが再生される。以降は静止ロゴに切り替わり、localStorageで管理する。

O L → A I

↑ クリックでアニメーション再生（初回起動を模擬）

アニメーションの意味

要素動き意味
Oの中の点（●）右に移動してAの中へOL（事務員）がAI化する
→（矢印）左→右にキラリと光が流れる変化・進化の演出
Aの中に点が現れるフェードインAIとして生まれ変わる
静止後OLAIのシンプルロゴ以降は静画で表示
// 初回起動チェック
const hasPlayed = localStorage.getItem('olai_logo_played');
if (!hasPlayed) {
  playLogoAnimation();
  localStorage.setItem('olai_logo_played', 'true');
}

// アニメーション：Oの点がAの中へ移動
// 矢印に光が左→右へ流れる（0.6s）
// Aの中に点がフェードイン（0.3s delay 0.7s）
// 完了後は静止ロゴに切り替え

LINE連携仕様

個別チャット（社長専用）

社長とOLAIの1対1チャット。全発言を処理する。見積・請求・顧客検索・支払い確認・現場管理など経営業務全般。

メンション専用設計の理由

グループ内では@OLAIメンションがある場合のみAPIを呼び出す。メンションなしの発言は完全スルー。APIコストを最小化する最重要設計。
// グループメッセージ処理
if (!message.text.includes('@OLAI')) {
  return; // APIを叩かない・コスト0
}
// メンションありの場合のみRouter Agentへ

const parseCompletion = (msg, unitType) => {

      return extractSquareMeters(msg);

DBスキーマ設計

-- 会社マスタ
CREATE TABLE companies (
  id                UUID PRIMARY KEY,
  name              TEXT NOT NULL,
  industry          TEXT,
  address           TEXT,
  phone             TEXT,
  tax_type          TEXT,          -- 'taxable'|'exempt'|'invoice'
  closing_day       INT,           -- 締め日（末日=31）
  payment_terms     TEXT,          -- 'next_month_end'など
  line_user_id      TEXT,
  stamp_image_url   TEXT,          -- 社印PNG（透過）
  logo_image_url    TEXT,
  pdf_template_json JSONB,         -- 書類レイアウト情報
  source_pdf_url    TEXT,          -- 元PDFの保存先
  plan_type         TEXT,          -- 'free'|'worker'|'business'
  created_at        TIMESTAMP
);

-- 職人マスタ
CREATE TABLE workers (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  name            TEXT NOT NULL,
  line_user_id    TEXT UNIQUE,
  bank_name       TEXT,
  bank_branch     TEXT,
  account_number  TEXT,
  is_boss         BOOLEAN DEFAULT false, -- 社長兼職人フラグ
  created_at      TIMESTAMP
);

-- 単価マスタ（職人×現場×元請けで異なる単価に対応）
CREATE TABLE worker_unit_prices (
  id            UUID PRIMARY KEY,
  worker_id     UUID REFERENCES workers(id),
  company_id    UUID REFERENCES companies(id),
  site_id       UUID REFERENCES sites(id),    -- NULL=デフォルト単価
  client_id     UUID REFERENCES clients(id),  -- NULL=全元請け共通
  unit_type     TEXT NOT NULL,
  -- 'room'|'sqm'|'daily'|'hourly'|'unit'|'fixed'
  unit_price    INT NOT NULL,
  valid_from    DATE NOT NULL,
  created_at    TIMESTAMP
);

-- 元請けマスタ
CREATE TABLE clients (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  name            TEXT NOT NULL,
  contact_name    TEXT,
  phone           TEXT,
  email           TEXT,
  address         TEXT,
  bank_name       TEXT,
  account_number  TEXT,
  closing_day     INT,
  payment_terms   TEXT,
  notes           TEXT,
  created_at      TIMESTAMP
);

-- 現場マスタ
CREATE TABLE sites (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  client_id       UUID REFERENCES clients(id),
  name            TEXT NOT NULL,
  room_format     TEXT,
  total_rooms     INT,
  line_group_id   TEXT,
  status          TEXT,           -- 'active'|'completed'
  created_at      TIMESTAMP
);

-- 完了記録
CREATE TABLE completions (
  id              UUID PRIMARY KEY,
  site_id         UUID REFERENCES sites(id),
  worker_id       UUID REFERENCES workers(id),
  room_number     TEXT,
  quantity        DECIMAL,        -- 部屋数・㎡・時間・個数
  unit_type       TEXT,
  unit_price      INT,
  completed_at    TIMESTAMP
);

-- ユーザー特典管理（無料化・割引）
CREATE TABLE user_benefits (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  benefit_type    TEXT,           -- 'free'|'discount'|'feature_unlock'
  discount_rate   INT,            -- 割引率（100=無料）
  feature_key     TEXT,           -- 特定機能のみ解放
  start_date      DATE,
  end_date        DATE,           -- NULLなら無期限
  reason          TEXT,
  created_by      TEXT,           -- 操作した管理者
  created_at      TIMESTAMP
);

-- 請求書送付ログ
CREATE TABLE invoice_deliveries (
  id              UUID PRIMARY KEY,
  invoice_id      UUID REFERENCES invoices(id),
  worker_id       UUID REFERENCES workers(id),
  company_id      UUID REFERENCES companies(id),
  sent_via_line   BOOLEAN DEFAULT true,
  sent_via_email  BOOLEAN DEFAULT true,
  email_address   TEXT,
  sent_at         TIMESTAMP,
  opened_at       TIMESTAMP   -- メール開封検知（将来）
);

-- 請求書マスタ
CREATE TABLE invoices (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  client_id       UUID REFERENCES clients(id),
  worker_id       UUID REFERENCES workers(id),  -- 職人発行の場合
  invoice_type    TEXT,          -- 'estimate'|'invoice'|'receipt'
  amount          INT,
  tax_amount      INT,
  total_amount    INT,
  due_date        DATE,
  status          TEXT,          -- 'draft'|'sent'|'paid'|'overdue'
  pdf_url         TEXT,
  period_from     DATE,
  period_to       DATE,
  created_at      TIMESTAMP
);

-- 仕訳テーブル
CREATE TABLE journal_entries (
  id              UUID PRIMARY KEY,
  company_id      UUID REFERENCES companies(id),
  date            DATE NOT NULL,
  description     TEXT,
  amount          INT,
  tax_amount      INT,
  account_type    TEXT,          -- '材料費'|'外注費'|'交通費'など
  receipt_image   TEXT,          -- 領収書画像URL
  created_at      TIMESTAMP
);
  id              UUID PRIMARY KEY,
  admin_name      TEXT,
  action          TEXT,
  target_company  UUID,
  detail          TEXT,
  created_at      TIMESTAMP
);
